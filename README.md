# Dolmen
A 2D Monogame game engine

FEATURES:

 - Map Editor
 - Asset Management
 - Game states
 - Camera
 - Debug display
 - Runtime Console
 - Mouse & Keyboard Input handling
 - Serialization
 - Basic networking (telemetry or score)
 - Config file

 GRAPHICAL:
 - Particle Effects
 - Shaders and Post Processing
 - Procedural Animations
 - UI Library
